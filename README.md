**SwiftShop** is a high-performance, aesthetically crafted e-commerce platform designed for the modern web. Built with a focus on speed, design precision, and user experience, it offers a premium shopping journey from discovery to checkout.

## ✨ Key Features

- **Modern UI/UX**: A minimalist, high-contrast interface with fluid transitions and layout animations powered by `motion`.
- **🌗 Native Dark Mode**: Fully themed for comfortable browsing in any lighting condition.
- **🤖 AI-Powered Intelligence**: Integrates Gemini AI to generate automated summaries of product reviews, helping customers make informed decisions faster.
- **🔍 Advanced Discovery**:
  - **Dynamic Sorting**: Filter products by price, popularity, or date.
  - **Quick View**: Preview product details without leaving the gallery.
  - **Breadcrumb Navigation**: Intuitive path tracking for easy site navigation.
- **🛒 Seamless Commerce**:
  - **Interactive Cart**: Real-time updates and sliding cart drawer.
  - **Guest Checkout**: Frictionless purchase flow with shipping validation.
  - **Product Comparison**: A specialized tool to evaluate specs side-by-side.
  - **Smart Wishlist**: Save items for later with persistent storage.
- **📱 Responsive & Fast**: Mobile-first architecture ensures a perfect experience on any device, optimized by the Vite build system.

## 🛠️ Tech Stack

- **Framework**: React 18
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Motion (formerly Framer Motion)
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Backend (Mock)**: Express for API orchestration
